Core Component:-

a. Signal Creation:
[Initialize Value] -> [Create Subscriber List] -> [Define Get/Set Methods]

b. Computed Creation:
[Define Computation] -> [Create Signal] -> [Set Up Auto-Recompute]

c. Effect Creation:
[Define Effect] .
-> [Run Initially] -> [Track Dependencies]

d. Dependency Tracking:
[Read Signal] -> [Current Effect?] -> [Yes: Add to Dependencies] -> [No: Just Return Value]

e. Signal Update:
[Set New Value] -> [Value Changed?] -> [Yes: Notify Subscribers] -> [No: Do Nothing]

f. Update Propagation:
[Notify Subscriber] -> [Is Computed?] -> [Yes: Recompute] -> [No: Run Effect]
-> [Notify Its Subscribers]

g. Batching (Optional):
[Multiple Updates] -> [Collect Changes] -> [Sort Topologically] -> [Apply All at Once]

h. Cleanup:
[Unsubscribe Effect] -> [Remove from Dependency Lists] -> [Garbage Collect if Unused]
